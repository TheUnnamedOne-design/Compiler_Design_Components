# LEX Programming Exercises

This directory contains LEX programs for various lexical analysis tasks.

## Program List

1. Write a LEX program to count the number of characters in a given input.

2. Write a LEX program to count the number of words in a sentence.

3. Write a LEX program to count the number of lines in a file.

4. Write a LEX program to count the number of vowels in the given input.

5. Write a LEX program to convert all lowercase letters to uppercase.

6. Write a LEX program to identify and count digits in the input.

7. Write a LEX program to identify and count keywords and identifiers.

8. Write a LEX program to recognize integers and floating-point numbers.

9. Write a LEX program to count the number of operators and operands.

10. Write a LEX program to remove single-line and multi-line comments from a C program.

11. Write a LEX program to recognize relational and logical operators.

12. Write a LEX program to count the number of uppercase and lowercase letters.

13. Write a LEX program to identify valid and invalid identifiers.

14. Write a LEX program to find and print all email IDs from a given text.

15. Write a LEX program to perform lexical analysis of a C program.

16. Write a LEX program to generate tokens for keywords, identifiers, constants, and operators.

17. Write a LEX program to implement a symbol table.

18. Write a LEX program to validate arithmetic expression.
